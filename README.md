# LTD CORP — Panneau de Gestion

Panneau de gestion interne pour GTA V Roleplay.

## 🚀 Déploiement GitHub Pages

1. Crée un nouveau dépôt GitHub (public)
2. Uploade le fichier `index.html` à la racine
3. Va dans **Settings → Pages → Source** : sélectionne `main` branch, dossier `/root`
4. Ton site sera disponible sur : `https://TON-USERNAME.github.io/NOM-DU-REPO/`

## 💾 Sauvegarde des données

Les données (comptes, ventes, stock) sont sauvegardées dans le **localStorage** du navigateur de chaque utilisateur.

- Chaque appareil/navigateur conserve ses propres données
- Les données persistent entre les sessions
- Pour partager les données entre plusieurs appareils, utilisez le même navigateur ou exportez via CSV

## 🔐 Connexion par défaut

Le compte administrateur initial est configuré dans le code.  
Un admin peut créer de nouveaux comptes depuis **Administration → Ajouter un compte**.
